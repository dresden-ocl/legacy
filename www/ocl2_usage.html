Dresden OCL2 Toolkit - Usage examples

How to use the Dresden OCL2 Toolkit

Here you can find a few usage examples to learn, how to work with the Dresden OCL2 Toolkit. Attention: These examples use the Dresden OCL2 Toolkit Release 1.2 which is outdated by now.

  • Screencast: Explanation of the code generator of the OCL Toolkit (will open in a new window)

    This example screencast written by Ronny Brandt explains how to use the JavaCodeGenerator

  • Tutorial & Screencast: How to use OCL22SQL (will open in a new window)

    This tutorial and screencast written by Michael Thiele and Claas Wilke explains how to use the tool OCL22SQL which generates SQL tables and views from a given UML/OCL model.
