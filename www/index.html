News

2006/06/12 - Release 1.1 of Dresden OCL2 Toolkit released

The team of the Dresden OCL Toolkit project is happy to announce the release of version 1.1 of the Dresden OCL2 Toolkit supporting OCL 2.0. The new version is available from the project download area at Sourceforge: http://sourceforge.net/project/showfiles.php?group_id=5840&package_id=173625

2005/12/22 - First Dresden OCL2 Toolkit Release

We are proud to present the first release of the Dresden OCL2 Toolkit. This release is a version of the Dresden OCL Toolkit supporting OCL2.0. You can obtain this release from our files section on SourceForge: https://sourceforge.net/project/showfiles.php?group_id=5840
