...snapshots

July 3, 2000

  • dresden-ocl.jar contains everything needed to run the ocl compiler, excepting xml4j.jar. Start with 
    java -cp dresden-ocl.jar:xml4j.jar tudresden.ocl.ConstraintEvaluation
  • ocl-argo.jar is to be used with Argo/UML. Is a subset of dresden-ocl.jar. Include it into your Argo/UML installation.
  • dresden-ocl-src.tar.gz: the complete source including test suites.
  • dresden-ocl-api.tar.gz: the javadoc api documentation, also available online.
  • jocl.jar: the prototype tool JOcl is available in a separate package.

"Corrupt JAR file" when downloading with netscape? Select "Save Link As.." in the context menu, instead of just clicking the link.

The ultimate source is the cvs repository at sourceforge.

...dependencies

These download are offered as a convenience only (and because the compiler is tested with these versions); for more recent versions, visit the original websites.

  • xml4j.jar: XML4J version 2.0.4 by IBM Alphaworks, needed for dresden-ocl.jar, but not for ocl-argo.jar.
  • test.jar: JUnit version 1.0 by Extreme Programming, needed for compiling and running the test suite. The test suite is available in the source package only.

"Corrupt JAR file" when downloading with netscape? Select "Save Link As.." in the context menu, instead of just clicking the link.
