Introduction

This is a tutorial where you will learn how to work with the tool OCL22SQL.

It is divided into three parts. First of all you will learn something about example UML and OCL models which are used during this tutorial. Then you will read a short explanation of the transformation rules the tool is working with. The third and last part of the tutorial is an animation which will guide you through the usage of the OCL22SQL tool.

If you don't need to know all background information you may start the example animation directly.
