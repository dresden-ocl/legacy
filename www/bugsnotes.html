Bugs & Notes

Notes

Dresden OCL2 Toolkit

  • Flat namespace inside scopes: overloaded operations or operations and attributes with the same name inside the same namespace are not supported by the parser. This is a (baffling) requirement in the OCL2.0 specification, see ch. 4.4.1
  • Parser supports only UML15 models. Parser could probably be adapted to MOF models (later).

This list is by no means exhaustive!
