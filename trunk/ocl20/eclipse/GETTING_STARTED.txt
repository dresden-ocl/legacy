Instructions for the Dresden OCL2 Toolkit Version 2
===================================================

The Dresden OCL2 Toolkit Version 2 is organized as a group of Eclipse Plugins.

How to run the Dresden OCL2 Toolkit Version 2
=============================================

The Dresden OCL2 Toolkit is distributed as source code and binary jar archives. How to run the toolkit
in both situation is explained below.

a) How to run a source code distribution

If you got the source code distribution, import the directories located in the directory 'eclipse' 
as plug-ins into your workspace. Afterwords you can run the plug-ins in a new Eclipse instance.

b) How to run a binary distribution

If you got a binary distribution, copy the jars located in the directory 'eclipse/plugins' into the
'plugins' directory of your eclipse distribution.


For further details look into the tutorials provided in the directory 'doc'.

More information about the Dresden OCL2 Toolkit is also available at the Toolkit Website
(http://dresden-ocl.sourceforge.net/).
