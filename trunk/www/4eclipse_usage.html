Dresden OCL2 for Eclipse - Usage examples

How to use Dresden OCL2 for Eclipse

Here you can find a few usage examples to learn, how to work with Dresden OCL2 for Eclipse.

Tutorials

  • An introduction into Dresden OCL2 for Eclipse

    This tutorial written by Claas Wilke and Ronny Brandt explains how to install Dresden OCL2 for Eclipse, how to load models and model instances and how to parse and interpret constraints.

  • How to use the Java Code Generator Ocl2Java

    This tutorial written by Claas Wilke explains how the Java code generator Ocl2Java of Dresden OCL2 for Eclipse can be used to generate AspectJ code for OCL constraints.

  • Examples provided with Dresden OCL2 for Eclipse

    This PDF provides a short overview over all examples which are provided with Dresden OCL2 for Eclipse.

  • Pivot Model Adapter Generation

    This tutorial written by Michael Thiele shortly explains how to generate a new Pivot Model Adapter using the Adapter Generator included into the toolkit.
