Dresden OCL2 for Eclipse - Usage examples

How to use Dresden OCL2 for Eclipse

Here you can find a few usage examples to learn, how to work with Dresden OCL2 for Eclipse.

Tutorials

All tutorials on Dresden OCL2 for Eclipse that have been provided at this website, are now combined and extended into one general manual of Dresden OCL2 for Eclipse.

  • Dresden OCL2 for Eclipse Manual

    The manual of Dresden OCL2 for Eclipse has been written by Claas Wilke and Michael Thiele. The manual contains tutorials that introduce into the installation and the use of Dresden OCL2 for Eclipse and also additional information for developers that want to use or extend Dresden OCL2 for Eclipse in their own tools and software.

Screencasts

  • Creating AspectJ code with the Java Code Generator Ocl2Java

    This screencast written by Claas wilke shortly explains how the Java code generator Ocl2Java of Dresden OCL2 for Eclipse can be used to generate AspectJ code for OCL constraints.

  • Adapting a domain-specific language with the Pivot Model

    This screencast written by Mathias Bräuer shows how a domain-specific model can be imported into Dresden OCL2 for Eclipse.
    Attention: Please note, that the screencast was not created with the last release of Dresden OCL2 for Eclipse. Thus, some menu and wizard names changed since the screencast's creation.

  • Generically modeling the OCL Standard Library

    This screencast written by Mathias Bräuer shortly explains how to the OCL standard library has been modeled using the Pivot Model of Dresden OCL2 for Eclipse.
