Dresden Toolkit for Eclipse - Usage examples

How to use the Dresden Toolkit for Eclipse

Here you can find a few usage examples to learn, how to work with the Dresden Toolkit for Eclipse.

Tutorials

  • An introduction into the Dresden Toolkit for Eclipse (will open as PDF file in a new window)

    The tutorial is also available in German!

    This tutorial written by Ronny Brandt and Claas Wilke explains how to install the Dresden Toolkit for Eclipse, how to load models and model instances and how to parse and interpret constraints.

  • Pivot Model Adapter Generation (will open as PDF file in a new window)

    This tutorial written by Michael Thiele shortly explains how to generate a new Pivot Model Adapter using the Adapter Generator included into the toolkit.

Screencasts

  • Generically modeling the OCL Standard Library (will open as screencast in a new window)

    This screencast written by Mathias Bräuer shortly explains how to model the OCL standard library using the Pivot Model.
