Bugs and Notes

Please use the bug tracker on SourceForge for known issues concerning the toolkit. You can also use the bug tracker to report new bugs.

Bugfixes in Different Releases

Please make sure that you use the last release of Dresden OCL2 for Eclipse and that the reported bug has not been fixed yet. To investigate the changes and bugfixes introduces with the different releases of Dresden OCL2 for Eclipse you may also read the changelog provided with Dresden OCL2 for Eclipse (available from SVN via this link).

Mailing list

Feel free to subscribe to the discussion mailing list on issues regarding the Toolkit. Additionally the mailing list's archive might contain helpful information.
