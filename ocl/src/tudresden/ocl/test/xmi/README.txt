
This directory contains the models used for the test suite.
See class tudresden.ocl.test.TestParser
It's basicly the same model constructed with several CASE tools,
differences are due to limitations of the CASE tools.

argo07:
-------
Argo/UML 0.7
see argouml.tigris.org.

argo08:
-------
Argo/UML 0.8
converted from the 0.7 model using MIMB XMI bridge
not yet used in test suite.
Contains a eps (Encaplulated PostScript) view of the model.

rose:
-----
Rational Rose 98i
exported with the Unisys plugin version 4.0.1? or so.
